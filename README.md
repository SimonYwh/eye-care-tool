# 护眼工具

智能护眼提醒工具，帮助你在长时间使用电脑时保护眼睛健康。

## 功能特性

- **用眼时长统计** — 实时记录连续用眼时间，达到阈值后自动提醒休息
- **屏幕亮度调节** — 一键调节显示器亮度，支持自定义亮度值
- **色温/夜间模式** — 暖色调护眼模式，减少蓝光对眼睛的刺激
- **系统托盘常驻** — 最小化到系统托盘，后台静默运行，不干扰工作
- **番茄钟提醒** — 可选的番茄工作法提醒，科学安排工作与休息
- **开机自启动** — 支持设置开机自动运行

## 安装

```bash
pip install -r requirements.txt
```

## 运行

```bash
python main.py
```

## 项目结构

```
├── main.py           # 程序入口
├── config.py         # 配置管理
├── core/             # 核心逻辑（亮度调节、色温控制等）
├── gui/              # 图形界面（系统托盘、设置窗口）
└── requirements.txt  # Python 依赖
```

## 依赖

- Python 3.7+
- screen_brightness_control — 屏幕亮度控制
- Pillow — 图像处理
- keyboard — 全局快捷键
