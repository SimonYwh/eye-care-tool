# 护眼助手

Windows 桌面护眼工具，通过显卡 gamma ramp 统一调节所有显示器的色温、亮度和色彩模式，适合长时间办公、阅读或夜间使用。

## 功能特性

- **色温调节**：支持 2700K–6500K，低色温偏暖、减少冷色刺激。
- **亮度调节**：支持 10%–100%，通过 gamma 曲线降低屏幕输出亮度。
- **内置预设**：夜间、护眼、日间、原始四种一键预设。
- **色彩模式**：支持正常、黑白、反色、淡色四种模式。
- **多显示器支持**：自动枚举所有显示器，并应用统一色彩配置。
- **平滑过渡**：切换预设和色彩模式时使用渐变过渡，减少突兀变化。
- **系统托盘常驻**：关闭窗口后最小化到托盘，可从托盘快速切换预设、显示窗口或退出。
- **设置持久化**：自动保存上次色温、亮度、预设和色彩模式。
- **安全恢复**：退出程序时自动恢复默认 gamma，避免残留屏幕效果。

## 系统要求

- Windows 10/11
- Python 3.10+
- 显卡/显示驱动支持 `SetDeviceGammaRamp`

## 安装

建议先创建虚拟环境：

```bash
python -m venv .venv
source .venv/Scripts/activate
pip install -r requirements.txt
```

## 运行

```bash
python main.py
```

启动后会显示主窗口，并在系统托盘创建图标。关闭主窗口不会退出程序，只会最小化到托盘；如需退出，请右键托盘图标选择“退出”。

## 使用说明

### 预设模式

| 预设 | 色温 | 亮度 | 适用场景 |
| --- | ---: | ---: | --- |
| 夜间 | 3400K | 70% | 夜间、暗光环境 |
| 护眼 | 4500K | 85% | 阅读、长时间办公 |
| 日间 | 5500K | 90% | 白天常规使用 |
| 原始 | 6500K | 100% | 恢复接近默认显示效果 |

### 色彩模式

| 模式 | 说明 |
| --- | --- |
| 正常 | 标准色彩，叠加色温与亮度调节 |
| 黑白 | 灰度显示，适合降低视觉干扰 |
| 反色 | 反转屏幕亮度输出，适合部分夜间阅读场景 |
| 淡色 | 低对比度、低饱和度的柔和显示效果 |

### 自定义调节

- 拖动“色温”滑块可在 2700K–6500K 之间调整，数值越低越暖。
- 拖动“亮度”滑块可在 10%–100% 之间调整。
- 黑白、反色、淡色模式下，色温滑块会禁用或不直接参与对应模式的显示效果。

## 配置文件

用户设置保存在：

```text
%APPDATA%\EyeComfort\settings.json
```

配置写入采用临时文件替换方式，降低程序中断导致配置损坏的风险。

## 项目结构

```text
├── main.py           # 程序入口和控制器
├── config.py         # 预设、范围和配置读写
├── core/
│   ├── gamma.py      # gamma ramp 构建、过渡和恢复
│   └── monitor.py    # Windows 多显示器枚举和 HDC 管理
├── gui/
│   ├── app.py        # CustomTkinter 主窗口
│   └── tray.py       # 系统托盘图标和菜单
└── requirements.txt  # Python 依赖
```

## 依赖

- customtkinter — 主窗口界面
- pystray — 系统托盘
- Pillow — 托盘图标绘制
- pywin32 — Windows 桌面环境支持

## 注意事项

- 本项目通过 gamma ramp 改变屏幕输出效果，不会修改显示器硬件亮度或系统亮度设置。
- 部分显卡驱动、远程桌面或 HDR/色彩管理环境可能限制 gamma ramp 生效。
- 如果程序异常退出导致屏幕效果未恢复，重新运行程序后点击“恢复默认”或从托盘退出即可重置。
